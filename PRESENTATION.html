03 // NETWORK INTERCEPTION

Chrome DevTools Protocol Deep Dive

browser/manager.py 
# Enable Chrome DevTools Protocol
cdp = await page.context.new_cdp_session(page)
await cdp.send("Network.enable")

# Capture every response WITH body
cdp.on("Network.responseReceived", handle)

async def handle(event):
    body = await cdp.send(
        "Network.getResponseBody",
        {"requestId": event["requestId"]}
    )
    if is_api_call(url):
        await store(Observation(url, body))

Why CDP?

Playwright's built-in page.on('response') doesn't give request body. CDP gives raw network access.

‚ö†Ô∏è Trade-off
CDP is Chrome-only. Chose coverage over portability.

08 // Q&A

Questions You Might Ask

Why LangGraph over AutoGen? ‚ñº
Fine-grained routing control. I can say "if confidence < 0.5, loop back". AutoGen is for agent conversations. I needed a state machine with conditional edges.
How do you avoid rate limiting? ‚ñº
Configurable delays, exponential backoff on 429s, guardrails limit max requests per domain. Aggressive exploration can still trigger blocks.
What's the LLM cost per session? ‚ñº
$0.10-0.50 with Gemini 2.0 Flash. ~2-3 LLM calls per iteration. Most tokens go to Navigator and Analyst.
Can it handle authentication? ‚ñº
Currently detects auth mechanisms (JWT, sessions, API keys) but doesn't bypass. Future: session cookie injection and OAuth automation.
